# 🏎️ Infinite Racer

An infinite lane-dodging racing game built with React + Vite.

## Run Locally

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build

```bash
npm run build
```

## Deploy to Vercel (free)

1. Push this folder to a GitHub repo
2. Go to https://vercel.com → New Project → Import repo
3. Vercel auto-detects Vite — just hit Deploy
4. Done! Live at `yourapp.vercel.app`

## Controls

- **← →** Arrow keys to switch lanes
- **Mobile**: Tap left/right side of screen
